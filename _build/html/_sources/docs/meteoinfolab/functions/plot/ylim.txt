*******************
ylim
*******************

.. function:: ylim(ymin, ymax)

    Set the *y* limits of the current axes.
    
    :param xmin: (*float*) Minimum limit of the y axis.
    :param xmax: (*float*) Maximum limit of the yaxis.
    
    **Example:**
    
    ::

        >>> ylim(2, 12)
